Steps need to run this project 

1. Clone the repo first
2. move to the directory

3. for backend
    cd backend
    1. npm i
    2. add .env file
       variables are :
       1. MONGO_URI : mongourl link
       2. JWT_SECRET : secret key
       3. PORT : 5000
      
    3. start command for backend
       npm run dev
   
  4.  for frontend
    cd frontend
    1. npm i
    2. start command for the frontend
      npm start
